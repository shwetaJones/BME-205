**Construct the Overlap Graph of a Collection of k-mers**

**Problem:**
	
	Given: A collection Patterns of k-mers.
	
	Return: The overlap graph Overlap(Patterns), in the form of an adjacency list.
	
**Sample DataSet:**
	
	ATGCG
GCATG
CATGC
AGGCA
GGCAT

**Sample Output:**
	
	AGGCA -> GGCAT
CATGC -> ATGCG
GCATG -> CATGC
GGCAT -> GCATG
